# MERN Author & Book Management REST API

##  Project Overview
This project is a RESTful API built using the MERN stack to manage Authors and their Books.

- One Author can have multiple Books
- One Book belongs to a single Author
- Built with Node.js, Express.js, MongoDB, and Mongoose
- Fully tested using Postman
